About the Project

NeuroFusion BioScan is a conceptual biomedical engineering project that combines MRI, Ultrasound, Photoacoustic Imaging and Artificial Intelligence to improve medical image analysis and assist disease detection.

Key Features

Medical Imaging

MRI + Ultrasound + Photoacoustic Imaging

Artificial Intelligence

Image Enhancement, Segmentation and Detection

Fusion 360

Complete CAD Model Design

Python

OpenCV Based Image Processing

System Workflow

Medical Images ↓ Image Processing ↓ AI Analysis ↓ Tumor Detection ↓ Visualization ↓ Report Generation

Technologies Used

  • OpenCV

  • Fusion 360
  • GitHub
